# Football Performance Analytics Dashboard

## Overview

This project presents an interactive Tableau dashboard designed to analyze football performance data across players, teams, and competitions. The dashboard transforms raw football statistics into actionable insights through interactive visualizations and performance metrics.

## Business Problem

Football clubs, analysts, and scouts rely on data-driven insights to evaluate player performance and team effectiveness. This dashboard helps users:

- Analyze player performance metrics.
- Compare teams across key statistics.
- Identify top-performing players.
- Explore trends through interactive visualizations.

## Tools Used

- Tableau
- Data Visualization
- Sports Analytics
- Data Cleaning

## Dashboard Features

### Player Analysis
- Goals scored
- Assists
- Minutes played
- Performance comparison

### Team Analysis
- Team rankings
- Offensive performance
- Defensive performance
- Match statistics

### Interactive Filtering
Users can filter by:
- Team
- Player
- Competition
- Season

## Key Insights

- Identifies top-performing players based on multiple performance metrics.
- Highlights team strengths and weaknesses.
- Supports data-driven decision making for scouting and performance evaluation.
